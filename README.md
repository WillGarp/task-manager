# 📋 Task Manager API

Uma API REST simples e eficiente para gerenciar tarefas, desenvolvida com **Spring Boot** e **Java**.

## 🚀 Tecnologias Utilizadas

- **Java 17+**
- **Spring Boot 3.2**
- **Spring Data JPA**
- **H2 Database** (em memória)
- **Maven**
- **Lombok**

## 📋 Funcionalidades

✅ Criar tarefas  
✅ Listar todas as tarefas  
✅ Listar tarefas pendentes  
✅ Listar tarefas concluídas  
✅ Buscar tarefa por ID  
✅ Atualizar tarefas  
✅ Marcar tarefas como concluídas  
✅ Deletar tarefas
